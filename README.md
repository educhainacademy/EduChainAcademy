# 🎓 EduChain Academy

> Learn. Build. Earn. Compliantly.

**EduChain Academy** is an open-source Web3 education platform teaching financial literacy, real estate entrepreneurship, and blockchain development — with utility token rewards designed for regulatory compliance.

## 🔐 Compliance-First Design
- ✅ Utility token (non-security) architecture
- ✅ KYC/AML integrated workflows
- ✅ Geo-fencing for restricted jurisdictions
- ✅ Transparent, auditable smart contracts
